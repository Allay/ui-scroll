Disabling

There is an extension of the adapter which allows to ignore user scroll/resize events. First you need an adapter: 
<div ui-scroll="item in datasource" adapter="myAdapter">{{item}}</div>
And then you are able to call disable/enable methods with appropriate logic: 
if(needToDisable) {
  myAdapter.disable();
}
else {
  myAdapter.enable();
}
Since .disable() is called no user scroll is being processed by ui-scroll engine.
User scroll processing is {{needToDisable ? "disabled" : "enabled"}}
